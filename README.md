

A web app that helps students report and recover lost student IDs on campus. Post about your lost ID, search for found items, and chat directly with other students to get your ID back.
Message the person who posted your ID to retrieve it back.

## What It Does

- **Post Lost Items** - Report your lost student ID with your name, ID number, and details about where you lost it
- **Search for Items** - Use the search bar to quickly find posts by student name or ID number
- **Message Directly** - Chat in real-time with the person who posted to arrange pickup or ask questions
- **Track Your Posts** - View all your posts and messages in one place

## Built With

### Backend
- **Spring Boot** - Handles the server and API
- **Spring Security + JWT** - Secure login and authentication
- **WebSockets** - Powers real-time messaging
- **PostgreSQL** - Stores posts, messages, and user data
- **Hibernate/JPA** - Manages the database

### Frontend
- **React + Vite**
